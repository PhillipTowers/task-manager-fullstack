# Task Manager Fullstack

Aplicación fullstack de gestión de tareas desarrollada como proyecto de portfolio.

## Tecnologías

### Frontend

* React
* TypeScript
* Vite

### Backend

* Node.js
* Express
* TypeScript
* Prisma

### Base de datos

* PostgreSQL (Neon)

### Autenticación

* JWT
* bcrypt

## Funcionalidades

* Registro de usuarios
* Inicio de sesión
* CRUD de tareas
* Edición de tareas
* Completar tareas
* Eliminar tareas
* Búsqueda de tareas
* Filtro por estado
* Contadores de tareas

## Arquitectura

Frontend:

* Components
* Hooks
* API Layer
* Types

Backend:

* Routes
* Controllers
* Services
* Prisma

## Deploy

Frontend: Vercel

Backend: Render

Base de datos: Neon
