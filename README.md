# Task Manager Fullstack

Aplicación fullstack de gestión de tareas con autenticación JWT y arquitectura cliente-servidor.

Permite registro de usuarios, inicio de sesión y gestión completa de tareas mediante una API REST desarrollada con Node.js, Express y Prisma.

Proyecto desarrollado como pieza de portfolio para demostrar competencias en desarrollo fullstack moderno.

---

## Demo

Frontend: https://task-manager-fullstack-rho.vercel.app

Backend: https://task-manager-fullstack-rho.vercel.app

La aplicación está desplegada utilizando servicios gratuitos de Vercel y Render.

El backend puede tardar algunos segundos en responder durante la primera solicitud si el servicio se encuentra inactivo debido a las limitaciones del plan gratuito.
